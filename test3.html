Set Theory Definitions

Set Definitions

Set: A set is simply a collection of unique items, organized neatly in curly brackets {}. No duplicates allowed!

Example: {apple, banana, cherry} represents a set of three distinct fruits.

Element: An element is an individual item in a set, represented by ∈ when we want to say it’s part of the set.

Example: If A = {1, 2, 3}, then 2 ∈ A means "2 is an element of set A."

Empty Set: The empty set (or null set) has no elements, so it’s represented by {} or ∅.

Example: {} or ∅ simply means an empty set.

Universal Set: The universal set U contains all elements within the context we’re working with, sort of like our “everything” set.

Example: If we’re looking at all integers, U would include all integers, positive, negative, and zero.

Subset: If every item in set A is also in set B, then A is considered a subset of B, denoted A ⊆ B.

Example: {1, 2} ⊆ {1, 2, 3} means every element in {1, 2} is found in {1, 2, 3}.

Proper Subset: A subset that doesn’t contain all elements of the other set. In other words, A ⊂ B means A is a subset of B but is not equal to B.

Example: {1, 2} ⊂ {1, 2, 3} (because {1, 2} is part of {1, 2, 3}, but not all of it).

Power Set: The power set of A includes all possible subsets of A, including the empty set and A itself.

Example: For A = {1, 2}, the power set P(A) = {{}, {1}, {2}, {1, 2}}.

Exploring Types of Sets

Finite Set: A set with a limited, countable number of elements.

Example: {red, green, blue} is finite because it has three specific elements.

Infinite Set: A set that goes on without end, like the set of natural numbers {1, 2, 3, ...}.

Equal Sets: Two sets that contain exactly the same elements. Order doesn’t matter, just the contents.

Example: {a, b, c} and {c, a, b} are equal.

Equivalent Sets: Sets that have the same number of elements, though the elements themselves might differ.

Example: {dog, cat, bird} and {red, blue, green} are equivalent because each has three items.

Singleton Set: A set with just a single element.

Example: {x} where x could be any element, but there’s only one.

Disjoint Sets: Sets that have no elements in common—no overlap.

Example: {1, 2, 3} and {4, 5, 6} have nothing in common, so they’re disjoint.

Overlapping Sets: Sets that share at least one common element.

Example: {1, 2, 3} and {2, 4, 6} overlap because they both contain 2.

Set Notations

Symbols and Notations in Set Theory

{} or ∅: This represents the empty set, which has no elements.

Example: {} or ∅.

∈: This symbol indicates that an item is in the set.

Example: 5 ∈ {3, 4, 5, 6}.

∉: This indicates that an item is not in the set.

Example: 8 ∉ {3, 4, 5, 6}.

⊆: A subset symbol, meaning all elements of A are also in B.

Example: {apple, banana} ⊆ {apple, banana, cherry}.

⊂: Proper subset, indicating A is a subset of B but not equal to B.

Example: {apple} ⊂ {apple, banana}.

∪: Union, combining all elements from two sets.

Example: {1, 2} ∪ {2, 3} = {1, 2, 3}.

∩: Intersection, representing elements common to both sets.

Example: {1, 2, 3} ∩ {2, 3, 4} = {2, 3}.

- or \: The difference, or what one set has that the other doesn’t.

Example: {1, 2, 3} - {2, 3} = {1}.

P(A): Power set of A, which includes every possible subset of A.

Example: For A = {x, y}, P(A) = {{}, {x}, {y}, {x, y}}.

Basic Proofs and Properties in Set Theory

Subset Proof

To prove that A ⊆ B, every element of A must also be in B.

Example: Given A = {1, 2} and B = {1, 2, 3}, since all elements of A are found in B, we conclude that A ⊆ B.

Set Equality

Two sets A and B are equal if and only if A ⊆ B and B ⊆ A.

Example: Sets {1, 2, 3} and {3, 2, 1} are equal because they contain exactly the same elements.

Idempotent Laws

  • Union Idempotent Law: A ∪ A = A (Uniting a set with itself does not change it).
  • Intersection Idempotent Law: A ∩ A = A (Intersecting a set with itself gives the same set).

Commutative Laws

  • Union: A ∪ B = B ∪ A
  • Intersection: A ∩ B = B ∩ A

Order doesn’t matter when uniting or intersecting sets!

Associative Laws

  • Union: (A ∪ B) ∪ C = A ∪ (B ∪ C)
  • Intersection: (A ∩ B) ∩ C = A ∩ (B ∩ C)

When combining three sets, it doesn’t matter which sets you group first.

De Morgan’s Laws

  • First Law: The complement of a union is the intersection of complements: (A ∪ B)' = A' ∩ B'
  • Second Law: The complement of an intersection is the union of complements: (A ∩ B)' = A' ∪ B'
