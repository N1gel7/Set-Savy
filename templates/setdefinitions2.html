Set Theory Definitions

Set Definitions

Set: A set is simply a collection of unique items, organized neatly in curly brackets {}. No duplicates allowed!

Example: {apple, banana, cherry} represents a set of three distinct fruits.

Element: An element is an individual item in a set, represented by ∈ when we want to say it’s part of the set.

Example: If A = {1, 2, 3}, then 2 ∈ A means "2 is an element of set A."

Empty Set: The empty set (or null set) has no elements, so it’s represented by {} or ∅.

Example: {} or ∅ simply means an empty set.

Universal Set: The universal set U contains all elements within the context we’re working with, sort of like our “everything” set.

Example: If we’re looking at all integers, U would include all integers, positive, negative, and zero.

Subset: If every item in set A is also in set B, then A is considered a subset of B, denoted A ⊆ B.

Example: {1, 2} ⊆ {1, 2, 3} means every element in {1, 2} is found in {1, 2, 3}.

Proper Subset: A subset that doesn’t contain all elements of the other set. In other words, A ⊂ B means A is a subset of B but is not equal to B.

Example: {1, 2} ⊂ {1, 2, 3} (because {1, 2} is part of {1, 2, 3}, but not all of it).

Power Set: The power set of A includes all possible subsets of A, including the empty set and A itself.

Example: For A = {1, 2}, the power set P(A) = {{}, {1}, {2}, {1, 2}}.
