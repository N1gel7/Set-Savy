Set Notations

{} or ∅: This represents the empty set, which has no elements.

Example: {} or ∅.

∈: This symbol indicates that an item is in the set.

Example: 5 ∈ {3, 4, 5, 6}.

∉: This indicates that an item is not in the set.

Example: 8 ∉ {3, 4, 5, 6}.

⊆: A subset symbol, meaning all elements of A are also in B.

Example: {apple, banana} ⊆ {apple, banana, cherry}.

⊂: Proper subset, indicating A is a subset of B but not equal to B.

Example: {apple} ⊂ {apple, banana}.

∪: Union, combining all elements from two sets.

Example: {1, 2} ∪ {2, 3} = {1, 2, 3}.

∩: Intersection, representing elements common to both sets.

Example: {1, 2, 3} ∩ {2, 3, 4} = {2, 3}.

- or \: The difference, or what one set has that the other doesn’t.

Example: {1, 2, 3} - {2, 3} = {1}.

P(A): Power set of A, which includes every possible subset of A.

Example: For A = {x, y}, P(A) = {{}, {x}, {y}, {x, y}}.
